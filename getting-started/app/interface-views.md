# Interface Views

The Trap Management Interface simplifies monitoring and controlling traps through three main modes: **Feed**, **Multi**, and **Map**, each offering unique views and functionalities.

### Feed Mode

The default page upon accessing the interface. It allows users to interact with the primary trap, control its settings, and view essential data quickly.

### Multi Mode

Enables viewing live feeds from all cameras on connected traps simultaneously, ideal for real-time, multisite monitoring.

### Map Mode

Displays all connected traps on a map, showing their geographical locations and enabling users to zoom out to see all traps at once for quick status checks.
